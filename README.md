# Online Notes App

A simple online notes application built using HTML, CSS, and JavaScript.
Notes are stored in the browser using LocalStorage.

## Features
- Create new notes
- Edit existing notes
- Delete notes
- Data persists using LocalStorage

## Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage
